# WFBarn

基于 Compose Multiplatform 构建的个人财务管理系统。

## 功能
- **多资产追踪**：股票、基金、现金、加密货币等。
- **每日盈亏**：记录每日波动，精准把控财富。
- **资金流水**：记录收入与消费，支持预算管理。
- **宏观分析**：自定义指标与趋势可视化。
- **云端同步**：支持 WebDAV 双端同步。

## 运行
```bash
./gradlew run
```

## 数据
- Windows: `%USERPROFILE%\Documents\WFBarn\state.json`
- Android: 应用私有目录
