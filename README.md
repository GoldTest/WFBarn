# 谷仓 (WFBarn)

Compose Multiplatform 个人财务管理工具。

## 最新更新 (v0.3.3)
- **小部件优化**：精简了“预算进度”小部件的布局，减少空白区域，信息展示更紧凑。
- **版本对齐**：统一了各端的版本号。

## 历史更新 (v0.3.2)
- **品牌升级**：正式更名为“谷仓”，寓意积蓄与丰收。
- **视觉优化**：全新红火色调文字图标，上移趋势箭头，提升品牌调性。
- **多端同步**：桌面端与 Android 端品牌显示深度统一。

## 核心特性
- **资产追踪**：支持多种资产类别。
- **财务记录**：每日盈亏、资金流水管理。
- **数据同步**：支持 WebDAV 云端备份与双端同步。

## 快速开始
```bash
./gradlew run
```

## 数据存储
- Windows: `Documents/WFBarn/state.json`
- Android: Internal Storage
